----
## Setup
1. Download ---- from GitHub
2. Install latest [Node.JS](https://nodejs.org/en/download/)
3. Open Terminal and Navigate to Project Directory
4. Run `$ npm install`
5. Run `$ npm install -g mocha`
6. Run Tests `$ npm test`
